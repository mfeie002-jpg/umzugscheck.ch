## ⚠️ Important: Hero-Integrated Variants (I-M, Q)

Variants I-M and Q are **"Hero-Integrated"** - they show trust signals **inside the hero section**, not below it.

**To see the full effect of these variants:**
1. Set **Homepage Hero = A** (HeroVariantOriginal)
2. The trust logos will appear inside the hero's CTA card
3. Below the hero, only a minimal stats strip is shown

If you use **Hero = C (SmartRouter)**, variants I-M and Q will only show the minimal strip because the SmartRouter hero doesn't have the integrated trust placement.
