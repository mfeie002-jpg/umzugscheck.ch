## 🔄 CONVERSION TRACKING SETUP (GA4)

### Step 1: GA4 Conversion Event
```javascript
// In your /pages/paid/ChampionQuickFlow.tsx:
// When form submits:

gtag('event', 'lead_submit', {
  'event_category': 'funnel',
  'event_label': 'champion_quick_flow',
  'value': 1,
  'currency': 'CHF'
});
```

### Step 2: Link GA4 to Google Ads
- Go to: Google Ads → Tools → Conversions
- Click "+" → Import → Select GA4 event "lead_submit"
- Set value: CHF 60 (estimated lead value)
- Assign to appropriate campaigns

### Step 3: Enhanced Conversions for Leads
- Enable "Enhanced Conversions" in campaign settings
- Upload CRM data weekly (phone → booking confirmation)
- Google learns which leads → real bookings (offline tracking)

## 💡 PRO TIPS

1. **Start narrow, expand wide**
   - Week 1: 5 exact keywords
   - Week 2: Add phrase match if CPL good
   - Week 3: Add broad if you can control negatives

2. **Budget is your friend**
   - Better to scale slowly (CHF 300/week) than blow CHF 5k on bad keywords
   - Once you hit CPL target, increase budget = proportional leads

3. **Keyword quality > Keyword volume**
   - "umzugsofferte" (1000 searches/month) beats "umzug tipps" (5000 searches/month) for conversions
   - Focus on "offerte", "angebot", "kosten", "preis" = buyer intent

4. **Ad copy is 70% of success**
   - Test 1 variable at a time (headline OR subheadline, not both)
   - Emotional vs. rational: "Schnell buchen" vs. "Fixpreis-Garantie"

5. **Landing page matters**
   - /champion-quick-flow = fastest conversion (use for high-intent)
   - /champion-trust-first = confidence builder (use for cold traffic, B2B)
   - Always include UTM params so you can attribute offline bookings
