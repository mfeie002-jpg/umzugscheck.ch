# Visual Assets Brief – Feierabend Umzüge

## 🎯 Project Overview

**Brand**: Feierabend Umzüge (Swiss moving company, since 1980)
**Visual Style**: Modern, clean, Swiss quality, warm but professional
**Inspiration**: 0123.com (Japan/USA) – branded trucks, real people, multi-channel visibility
**Color Palette**: Navy blue (#2E5A8F), Sunset orange (#FF6B35), White, Grays
**Purpose**: High-converting performance marketing assets (Google Ads, website, social media)

---

## 📸 Photography Requirements

### Style Guidelines

**Overall Aesthetic**:
- Natural light, warm tones (avoid harsh flash)
- Shallow depth of field (subject sharp, background softly blurred)
- Swiss locations (recognizable architecture, mountains, lakes)
- Real people, diverse (families, singles, seniors, professionals)
- Clean, organized, professional (not chaotic)

**Colors to Emphasize**:
- Branded items: Navy blue + orange (trucks, uniforms, boxes)
- Neutral backgrounds: White walls, natural wood, Swiss architecture
- Avoid: Red, yellow, green (competitors' colors)

**Mood**:
- Reassuring (not stressful)
- Competent (not amateur)
- Warm (not corporate-cold)
- Swiss (quality, punctuality, discretion)

- JPG (for print): 300 DPI, CMYK color space

**Variations**:
- Full color (navy + orange)
- Single color (navy only)
- White version (for dark backgrounds)
- Monochrome (grayscale)

**Logo Lockups**:
1. Logo + "Feierabend Umzüge" text (horizontal)
2. Logo + text (stacked/vertical)
3. Logo only (icon)

**Usage Guidelines**:
- Minimum size: 120px width (web), 20mm (print)
- Clear space: Minimum 0.5x logo height around logo
- Do not: Rotate, distort, change colors outside brand palette

## 📐 Technical Specifications

### File Formats

| Asset Type | Primary Format | Fallback | Notes |
|------------|----------------|----------|-------|
| Photos | WebP | JPG | 80% quality, progressive |
| Logos | SVG | PNG | Transparent background |
| Icons | SVG | PNG | 24x24, 32x32, 48x48 |
| Social | PNG | JPG | sRGB color space |

### Naming Convention

```
feierabend-{category}-{description}-{dimension}.{format}

Examples:
feierabend-hero-zurich-1920x1080.webp
feierabend-truck-branding-side-800x600.jpg
feierabend-logo-full-color.svg
feierabend-social-quote-1080x1080.png
```

### Optimization

- **Photos**: TinyPNG, Squoosh (target < 200KB per hero image)
- **Logos/SVG**: SVGOMG
- **Batch processing**: ImageMagick for resizing
