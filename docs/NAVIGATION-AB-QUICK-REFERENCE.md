# 17 Navigation Variants - Quick Reference

## 🎯 **Wähle deine Variante basierend auf dein Ziel:**

### **Für MAXIMUM CONVERSIONS:**
- `variant-f` (6) - Conversion-Killer: "bis 40% sparen"
- `variant-j` (10) - Final: "3-5 Offerten in 24-48h"
- `variant-d` (4) - Strategic: "Kosten & Planung zuerst"

### **Für MOBILE-FIRST:**
- `variant-m` (13) - 4 Punkte, Du-Form: "Plane DEINEN Umzug"
- `variant-o` (15) - 4 Punkte, emotional: "Hol dir gratis Offerten"
- `variant-e` (5) - Kurz & SEO: "Kosten berechnen"

### **Für SEGMENT-TARGETING:**
- `variant-h` (8) - Zielgruppen-Split: "Privatumzug" vs "Firmenumzug"
- `variant-g` (7) - Journey-Split: "Umzug planen" vs "Umzugsfirma finden"
- `variant-c` (3) - Offer-First: Schnell-Entscheider

### **Für PREMIUM/BRAND:**
- `variant-17` (17) - NavigationV17 mit Custom Design
- `variant-l` (12) - Best-of-Breed: Alle Best Practices
- `variant-b` (2) - Lovable: Action-First mit Verben

### **Für MINIMALISMUS:**
- `variant-k` (11) - Simpel & Clean (6 Punkte, 1-Wort Labels!)
- `variant-i` (9) - Mega Menu: Übersicht statt Komplexität

## 📱 **Mobile vs Desktop Performance**

```
BEST FÜR MOBILE:
→ variant-m (13) - 4 Punkte, kurz & kompakt
→ variant-o (15) - 4 Punkte, emotional
→ variant-e (5)  - Mobile-First optimiert
→ variant-k (11) - 6 Punkte aber sehr simpel

BEST FÜR DESKTOP:
→ variant-i (9)  - Mega Menu braucht Platz

```

## 🎯 **A/B Testing Empfehlung**

### **CONTROL (Baseline):**
```
variant-ultimate (1) oder variant-n (14)
→ Klassisch, bewährt, neutral
```

### **TREATMENT 1 (Mobile-First):**
```
variant-m (13) oder variant-o (15)
→ 4 Punkte, Du-Form, emotional
→ Sollte besonders auf Mobile gewinnen
```

### **TREATMENT 2 (Conversion-Killer):**
```
variant-j (10) oder variant-f (6)
→ Aggressive Benefits, Sales-Copy
→ Sollte CTR erhöhen
```

### **TREATMENT 3 (Zielgruppen):**
```
variant-h (8)
→ Privat vs Firma Split
→ Sollte Clarity für beide Segmente erhöhen
```

## ✨ **Besondere Highlights**

### **SWAPPED (andere Reihenfolge):**
```
Variante 3 (Concierge):  Offerten → Planung
Variante 4 (Strategic):  Planung → Offerten
→ Unterschiedliche Conversion Paths!
```

### **KOMPLETT ANDERS:**
```
Variante 8 (Zielgruppen-Split):
Punkt 1: "Privatumzug" (nicht Preisrechner!)
Punkt 2: "Firmenumzug" (nicht Umzugsfirmen!)
→ Segmentiert nach Kundentyp, nicht nach Journey!
```

### **NUR EIN WORT:**
```
Variante 11 (Simpel):
"Umzug" statt "Preisrechner"
"Umzugsreinigung" statt "Services"
→ Minimalistisch & scannable!
```

### **DU-FORM:**
```
Variante 13 & 15:
"Plane DEINEN Umzug" statt "Kosten & Planung"
→ Personal, emotional, mobile-optimiert!
```

### **4 PUNKTE:**
```
Variante 13 & 15:
Nur 4 Navigation Items statt 5
→ Schneller zu übersehen, Mobile-Friendly!
```

### **6 PUNKTE:**
```
Variante 11:
Extra Punkt für "Umzugsreinigung"
→ Hebt Service hervor!
```

## 🚀 **Quick Start**

```bash
# 1. Kopiere navigation-variants.ts
cp navigation-variants.ts YOUR_PROJECT/src/lib/

# 2. Nutze in deiner Nav-Komponente
import { NAV_VARIANTS } from '@/lib/navigation-variants';
const variant = NAV_VARIANTS[12]; // variant-m

# 3. Rendern
{variant.labels.preisrechner}
{variant.microcopy.preisrechner}
{variant.ctaCard.preisrechner.buttonText}

# 4. Testen
http://localhost:3000/?nav=variant-m
http://localhost:3000/?nav=variant-o
http://localhost:3000/?nav=variant-h
```
