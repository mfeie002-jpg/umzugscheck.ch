# TEIL 2: FEEDBACK-ANALYSE (7 Kategorien)

## 1. WORDING & COPY (Kritisch)

Analysiere JEDEN Text auf der Seite:

| Prüfpunkt | Frage |
|-----------|-------|
| Klarheit | Versteht man jeden Satz in <3 Sekunden? |
| Emotion | Spricht es das Herz an, nicht nur den Kopf? |
| Action | Sind CTAs stark genug? ("Offerte" vs "Jetzt sparen") |
| Swiss Tone | Seriös-schweizerisch, nicht marktschreierisch? |
| Konkret | Zahlen statt Floskeln? (553 CHF vs "hohe Marge") |
| Story Arc | Gibt es einen roten Faden beim Scrollen? |

**OUTPUT**: Gib 5 konkrete Rewrites für schwache Headlines/Texte.

---

## 2. VISUELLE GESTALTUNG (Kritisch)

| Prüfpunkt | Frage |
|-----------|-------|
| Hero Impact | Zieht der Hero sofort in den Bann? |
| Whitespace | Atmet das Design oder erdrückt es? |
| Hierarchie | Ist sofort klar, was wichtig ist? |
| Konsistenz | Gleiche Icons, Abstände, Farben? |
| Mobile | Touch-friendly? (48px Targets, Thumb-Zone) |
| Abwechslung | Monoton oder rhythmisch wechselnd? |

**OUTPUT**: 3 konkrete Design-Änderungen mit Skizzen/Beschreibungen.

---

## 3. EMOTIONALE WIRKUNG (Sehr Kritisch)

| Prüfpunkt | Frage |
|-----------|-------|
| Menschlichkeit | Sehe ich ECHTE Menschen, echte Momente? |
| Umzugs-Feel | Wird Stress UND Freude des Umziehens transportiert? |
| Trust | Fühle ich mich sicher? Genug Vertrauenssignale? |
| Begeisterung | Macht Lust auf mehr oder "nur informativ"? |
| Uniqueness | Fühlt sich anders an als Standard-Pitches? |

**OUTPUT**: Beschreibe das emotionale Erlebnis beim Scrollen in 3 Sätzen.

---

## 4. INFORMATIONSARCHITEKTUR

- Ist die Reihenfolge der Sections logisch?
- Gibt es Cognitive Overload irgendwo?
- Progressive Disclosure korrekt eingesetzt?
- Sticky-Nav hilfreich oder störend?

---

## 5. CONVERSION-OPTIMIERUNG

- Ist der primäre CTA offensichtlich?
- Wo könnte jemand abspringen? (Friction Points)
- Value Proposition in 5 Sekunden klar?
- Social Proof überzeugend platziert?

---

## 6. "MAKE IT ALIVE" – Lebendigkeit

| Was fehlt? | Vorschlag |
|------------|-----------|
| Animationen | Scroll-Reveal, Hover-Effekte |
| Interaktivität | Klickbare Elemente, Entdecken |
| Micro-Interactions | Feedback bei Actions |
| Video/Motion | Hero-Video, Demo-Clip |
| Dynamic Elements | Live-Counter, Progress-Bars |

---

## 7. DETAIL & POLISH

- Pixel-Perfect Alignment?
- Typography stark genug?
- Icons konsistent?
- Loading States vorhanden?
