4. FUNNEL PREFILL - Umzugsofferten.tsx
====================================

On mount, reads localStorage and prefills Step 1:

```tsx
useEffect(() => {
  try {
    const prefillRaw = localStorage.getItem('uc_prefill');
    if (prefillRaw) {
      const prefill = JSON.parse(prefillRaw);
      const age = Date.now() - (prefill.timestamp || 0);
      const MAX_AGE = 24 * 60 * 60 * 1000; // 24 hours
      
      if (age < MAX_AGE) {
        // Prefill Step 1 fields
        if (prefill.from) setFromLocation(prefill.from);
        if (prefill.to) setToLocation(prefill.to);
      }
    }
  } catch (e) {
    console.warn('Could not read prefill data');
  }
}, []);
```

5. KEY CONVERSION PRINCIPLES
====================================

Homepage = Commitment (minimal friction)
- Only ask From/To
- Show progress: "Step 1 of 4"
- Single CTA, no choices

Funnel = Decisions (detailed info)
- Ask rooms in Step 2
- Ask services in Step 3
- Ask contact in Step 4

Why this works:
- Reduces cognitive load on homepage
- User commits before making decisions
- Prefill creates momentum into funnel
- Progress indicator sets expectations
