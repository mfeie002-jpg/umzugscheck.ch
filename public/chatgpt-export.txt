ANALYTICS EVENT MAP (V3)
================================================================================

CANONICAL EVENTS:
- page_viewed (once per load)
- step_started (step, step_name)

- field_interacted (step, field key only - NO values)
- option_selected (step, option)

- name, email, phone
- exact address strings
- any freeform user text

CONSENT GATING:
- localStorage key: uc_consent
- Analytics events only fire when consent.analytics = true
- Debug overlay: localStorage.uc_debug = 'true'
