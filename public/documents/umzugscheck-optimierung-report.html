Mobile Keyboard Optimization

MultiStepCalculator.tsx
🎯 UX
⚡ Performance
❌ Vorher
<ValidatedInput
  type="tel"
  placeholder="+41 79 123 45 67"
  // Keine inputMode Angabe
/>

→ Auf vielen Geräten: Standard QWERTY-Tastatur
→ Nutzer muss manuell zu Zahlen wechseln
✅ Nachher
<ValidatedInput
  type="tel"
  inputMode="tel"
  placeholder="+41 79 123 45 67"
/>

→ Zeigt automatisch numerische Tastatur
→ Optimiert für Telefonnummern (+, -, Leerzeichen)

💡 Warum diese Änderung?

  • Richtige Tastatur: inputMode="tel" zeigt Telefon-Tastatur auf Mobile
  • Schnellere Eingabe: Keine manuelle Umschaltung nötig
  • Weniger Tippfehler: Grössere Zahlen-Tasten = präzisere Eingabe
  • Bessere Accessibility: Screenreader erkennen Feld-Typ korrekt
